Kinds of Databases

  1. Relational (SQL: Postgres, MySQL, SQLite)
  2. NoSQL
    • Document (JSON: Firebase, MongoDB)
    • Graph (Neo4j, ArangoDB)
    • Columnar (RedShift, Cassandra, ClickHouse)
    • Key-Value Store (DynamoDB, Couchbase, Redis)
    • Time Series (Prometheus, InfluxDB, TimescaleDB)

API tokens & secrets

Don't push your secrets or tokens to Github

Create a .env file in your project root & then .gitignore those files so they stay only on your local computer

Import the secrets/tokens into your project securely (usually with dotenv pkg or for Vite.js using import.meta.env.VITE_SECRET_NAME

Deploying

  1. in your Github repo, go into the Settings tab, and click on "Pages"
  2. Under "Build and deployment", set your branch to your main/master/central branch, and set the folder to "/docs" instead of "/(root)"
  3. Update your Vite project's vite.config.js to look like the following: 
    import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  base: "/YOUR-REPO-NAME",
  build: {
    outDir: 'docs'
  }
})
  4. Build your project by running npm run build
  5. Git add your files and push them up to Github (make sure you are on the branch specified in your Github Pages settings)
  6. Wait a few moments for Github to run its checks and deploy!
