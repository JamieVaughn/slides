API tokens & secrets

Don't push your secrets or tokens to Github

Use .env files and then .gitignore those files so they stay only on your local computer

Import the secrets/tokens into your project securely (usually with dotenv pkg or for Vite.js using import.meta.env.VITE_SECRET_NAME

Project Ideas

  • (re)Design your own personal portfolio website
  • Build out an existing product/site your own way: soundcloud.com, goodreads.com, Slack chat, etc.. )
  • Build a browser extension that makes your life simpler
  • Design some sort of framework to build projects on top of (like our MVC but better!)
  • Build a web scraper to collect data from a certain site.
  • Create a Reddit or Discord bot! See /r/RequestABot for ideas, and look at /r/botwatch for help and inspiration.
  • If you're more inspired by hardware projects, then get a Raspberry Pi, set up Arch Linux, make it accessible over the web via SSH, connect up sensors to measure temperature, light, etc. & display the output.
